Creating a Component

Angular apps are built with Typescript, HTML, and CSS

Components are fundamental building block of the application.

How to build a component in Angular?

  1. Import Component decorator in the file level
  1. Implement decorators (allows to take properties or class and add behavior to it)

    The Component decorator takes few parameters like selector which is like a name given to it

    example: 

    // importing "Component" decorator in file level 
import { Component } from '@angular/core'

// implementing "Component" decorator and adding behavior to it
@Component({
	selector: 'app-root', // name by which it will be used as  
	standalone: true, // keeps modules option as optional and not required 
	template: `<h1> Hey, Frontend Masters! </h1>` // could be a .html file
	styles: `h1 { color: red }` // could be .css file. Styles are scoped within the component file only 
})

// implemnting component class. Can be empty class if no logic is present 
// the component class can function standalone without decorator too if not required 
class AppComponent{}

Dynamic values in components

documentation of interpolation in angular: https://angular.dev/guide/templates/interpolation/

It is done with string interpolation. A valid expression can be dynamically rendered which can be variable name to calculations as well.

A variable is defined in the typescript file and then can be used in HTML

example: 

import { Component } from '@angular/core';
import { CommonModule } from '@angular/common';
import { RouterOutlet } from '@angular/router';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [CommonModule, RouterOutlet],
  // dynamically rendering the variable with string interpolation
  template: ` <p>username: {{ userName }}</p> `,
  styles: ``,
})
export class AppComponent {
  // creating a variable 
  userName = 'codingChamp';
}

Component Composition

To use a component:

  1. Import the component file
  1. Import the component in the Component decorator
  1. Reference it in the template

example: 

// Import the component file 
import { UserInfoComponent } from "./user-info.component'; 

@Component({
	// Import the component in the Component decorator
	imports: [UserInfoComponent ], 
	// Reference imorted component in the template 
	template: `
		<section> 
			<app-user-info />  
		</section> 
  `
})
